# **App Name**: GrepX Corporate AI Gateway

## Core Features:

- Strategic Content Presentation: Comprehensive overview of GrepX's mission, vision, and offerings, designed to attract clients, talent, and investors.
- Interactive AI Concept Explainer Tool: Interactive tool where users can click on or type AI terms to see explanations, generated by a self-hosted LLM, appearing in a tooltip or modal.
- AI Use Case Idea Sparker Tool: Interactive module where users select an industry and input a business challenge to generate AI application ideas, using a self-hosted LLM to reason about possible applications.
- Responsive Design: Fully responsive layout that adapts seamlessly across desktops, tablets, and mobile devices, ensuring optimal viewing experience.
- Performance Optimization: Prioritize fast loading times, smooth transitions, and efficient resource usage for optimal performance, considering the self-hosted environment.
- Security Implementation: Ensure the website adheres to security standards, including SSL/TLS encryption, input validation, and protection against common web vulnerabilities.
- Accessibility Standards: Adherence to WCAG 2.1 Level AA for keyboard navigability, screen reader compatibility, and sufficient color contrast to comply with basic accessibility guidelines.

## Style Guidelines:

- Primary color: Saturated teal (#4DB6AC) to convey innovation and trust in the tech industry.
- Background color: Light gray (#F0F4F3) to create a clean, modern, and professional look.
- Accent color: Deep purple (#7E57C2) for interactive elements, call-to-action buttons, and highlights, offering a strong contrast and conveying sophistication.
- Clean and modern typography choices for headers and body text to enhance readability.
- Use a consistent style for icons throughout the website.
- Modern, intuitive layouts that emphasize key content and encourage user exploration.
